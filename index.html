About

Hi! I'm Sofia, a Web Developer in the Washington, DC Metro Area. I've been a project manager in various industries for the past 7 years but am now eagerly making a career change into web development. I fell in love with coding as a project manager for a Software Development firm. I later had a chance to practice coding while helping my land development team use GIS (Global Information System) to better track their housing development.

During that time, I learned just how fascinating and fun coding is! I am now graduating from GW's Coding Bootcamp and look forward to this exciting new career.
Technologies I have learned: Node.js, Javascript, jQuery, React, GIT, GitHub, mySQL, Firebase, HTML, CSS, Bootstrap, Media Queries, APIs, JSON, AJAX, the Command Line, GIS (Global Information Systems)

GW Cards Against Humanity

This is a cards against humanity computer game. It uses Javascript, HTML, CSS, Handlebars, Firebase, Heroku and mySQL. The game plays with 4 people. User one is assigned to be Czar at the beginning of the game. It also tracks points, username, and dealt hand, but it doesn't currently allow users to play a second round. I am working on updating it!
